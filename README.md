This template:
```pug
body
  :jsx
    /** @jsx dom */
    var dom = React.createElement;

    ReactDOM.render(
      <h1>Hello world!</h1>,
      document.getElementById('main')
    );
```

will be rendered like this:
```html
<body>
  <script type="text/javascript">
    "use strict";

    /** @jsx dom */
    var dom = React.createElement;

    ReactDOM.render(dom(
        "h1",
        null,
        "Hello world!"
    ), document.getElementById('main'));
  </script>
</body>
```
